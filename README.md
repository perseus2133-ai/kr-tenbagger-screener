# 🇰🇷 KR Tenbagger Screener

코스피·코스닥 상장 주식 중 **3년 뒤 매출·영업이익이 2배 이상 성장할 것으로 추정되는 중소형 텐배거 후보**를 자동 스크리닝하는 Streamlit 대시보드.

## 스크리닝 조건 (기본값)

1. 현재 시가총액 **< 1조 원**
2. 3년 뒤 추정 영업이익 **≥ 2,000억 원**
3. 3년 뒤 추정 매출·영업이익이 기준연도 대비 **2배 이상**

> 추정 방식: 네이버 금융 「기업실적분석」 최근 연간 매출·영업이익으로 CAGR 산출 → 3년 외삽.

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 데이터 소스

- 종목 리스트: `FinanceDataReader`
- 시가총액: `pykrx`
- 재무 (매출·영업이익): 네이버 금융 크롤링

## 배포

Streamlit Community Cloud — `app.py`, Python 3.11 권장.

## ⚠️ 면책

본 도구의 출력은 **투자 참고용**이며, CAGR 단순 외삽 기반 추정이라 한계가 큽니다. 투자 결정의 책임은 사용자 본인에게 있습니다.
